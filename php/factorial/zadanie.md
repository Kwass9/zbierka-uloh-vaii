## Zadanie
Vytvorte skript v jazyku PHP, ktorý bude schopný zobraziť a vypočítať [faktoriál](https://sk.wikipedia.org/wiki/Faktori%C3%A1l) pre čísla od 0 po 10. Výsledky zobrazte nasledovne:

```html
<ul>
    <li>0! = 1</li>
    <li>1! = 1</li>
    <li>2! = 2</li>
    <li>3! = 6</li>
    <li>4! = 24</li>
    <li>5! = 120</li>
    <li>6! = 720</li>
    <li>7! = 5040</li>
    <li>8! = 40320</li>
    <li>9! = 362880</li>
</ul>
```
